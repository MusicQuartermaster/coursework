The following is a C++ program I wrote for one of my courses in the Fall 2020 semester. It replicates the PEP8 Simulator, which is software that simulates how the processor interprets and executes Assembly code. After each operation, it prints out the state of the processor registers (accumulator, index, stack pointer, and program counter). The current pep8 object code loaded in data.txt prints out the word "Hi"
